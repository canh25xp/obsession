# Obsession

An interactive date invitation web app — because "No" is not an option.

## What Is This?

A playful, single-page app that asks the question: **Wanna go see a movie with me?**
The catch? The "No" button runs away, the "Yes" button keeps growing, and the background gets darker with every attempt to decline.
Eventually, a pleading video takes over.
The only way out is to say **Yes** and pick a date.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to try it.

| Command         | Description              |
| --------------- | ------------------------ |
| `npm run dev`   | Start development server |
| `npm run build` | Production build         |
| `npm run start` | Start production server  |
| `npm run lint`  | Run ESLint               |
